Ideen: Tower Defense PVP Sieg bei zerstörung des Gegn. Towers/ wenn dieser keine Bauarbeiter und Gebäude mehr hat

Turn based RTS 
DEFENSE:
Bauarbeiter Character
Gebäude können nur von Bauarbeiter gebaut werden 
Gebäude für Eco, zerstörung gegn. Truppen, verteidigung (Bspw. Miene, Armbrust, Mauer/ Sonnenblume, Erbse, Wallnuss)
Bauarbeiter können getötet werden
Neue Bauarbeiter Kaufbar

ATTACK :
Wahl aus beschwörbaren Angreifern				
	verschieden Prioritäten,  was angegriffen wird
Buffs und Angriffsspells (Mehr HP für Kobolde, Rakete die auf gewählte stelle fliegt)
